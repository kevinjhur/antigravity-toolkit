# Antigravity Default Features Overview

Antigravity is an agent-first Integrated Development Environment built on top of VS Code. It ships with a comprehensive suite of built-in features that elevate the coding experience from autocomplete to autonomous, multi-agent development.

## Core Capabilities

- **Agent Manager**: Orchestrates multiple autonomous agents working in parallel.
- **Planning Mode**: A structured workflow where agents research, propose an `implementation_plan.md`, await user approval, execute while tracking progress in `task.md`, and summarize in `walkthrough.md`.
- **Knowledge Items (KI)**: A system for curating and recalling repository-specific context, past solutions, and architectural patterns.
- **Model Context Protocol (MCP)**: First-class support for MCP, acting as the "USB-C port for AI" to securely connect agents to external tools, databases, and APIs.
