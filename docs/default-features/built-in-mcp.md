# Built-in MCP Servers

Antigravity natively integrates with several default Model Context Protocol (MCP) servers. These are configured in `mcp_config.json`.

## Recommended Default Servers

- **context7**: For resolving library IDs and querying official documentation.
- **sequential-thinking**: Enables Chain-of-Thought (CoT) reasoning for complex tasks.
- **TestSprite**: Automated test generation and execution, frontend/backend test plans, and dashboard integration.
- **supabase**: Interacts with Supabase databases.
- **stitch**: Generates UI designs, screens, and extracts design contexts.
- **mermaid-renderer**: Renders Mermaid diagrams into PNGs for viewing.
- **notebooklm-mcp**: Interfaces with Google's NotebookLM for advanced research, note-taking, and querying notebooks.
- **chrome-devtools-mcp**: Headless browser automation (click, fill form, take screenshot, lighthouse audit, performance traces).
- **ollama**: Local LLM management (pull, push, chat completion, etc.).
- **Confluence**: Integration for corporate wikis and documentation.

*Note: For setup and installation instructions for these servers, refer to the [MCP Setup Guide](../mcp-guide.md).*
